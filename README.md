# Practice Team App

シンプルな自己紹介ページのプロジェクトです。

## 概要

HTML と CSS だけで作られた、自己紹介ページ。
チーム開発のワークフロー練習用のリポジトリです。

## セットアップ

クローンして、`index.html` をブラウザで開くだけで動きます。

```bash
git clone https://github.com/norino28/practice-team-app.git
cd practice-team-app
# index.html をブラウザで開く
```

## ファイル構成

- `index.html` - メインページ
- `style.css` - スタイル定義

## 開発ルール

- main ブランチへの直 push は禁止
- 修正は feature ブランチで作業し、PR を出す
- PR には変更内容と動作確認方法を記載
